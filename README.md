# Email Spam Classifier

This project is a Machine Learning based Email Spam Classifier built using Python and Streamlit.

## Features

* Classifies messages as Spam or Not Spam
* Built using NLP techniques (TF-IDF)
* Deployed using Streamlit Cloud

## Tech Stack

* Python
* Scikit-learn
* Streamlit
* Pandas, NumPy

## How to Run

1. Install requirements
2. Run: streamlit run app.py
